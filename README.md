## 本软件特别适合在电脑前工作的人

### 1. 秒表
### 2. 发正念提前1分钟全屏弹窗和响铃，到点敲钟。
### 3. 倒计时，比如洗衣服后设倒计时60分钟，到点全屏弹窗和响铃，免得忘了衣服在洗衣机；煮鸡蛋、煮面也可以设倒计时
### 4. 可以循环播放自己任选的音频（本人没找到这样的播放器），这个特别时候无限循环播放《九评共产党》、《解体党文化》等等

## 本软件关闭后常驻系统托盘，右键系统托盘图标，可以选 退出/显示原窗口
